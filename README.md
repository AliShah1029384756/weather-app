# Weather App

A clean weather app that shows real-time weather data for any city using the OpenWeatherMap API.

## Features
- Search weather by city name
- Current temperature with °C / °F toggle
- Weather condition icon and description
- Feels like, humidity, wind speed, visibility
- Error handling for invalid cities or network issues

## Setup & API Key

1. Sign up for a free account at [openweathermap.org](https://openweathermap.org/api)
2. Copy your API key from the dashboard
3. Open `app.js` and paste your key on **line 2**:
   ```js
   const API_KEY = 'your_key_here';
   ```
4. Open `index.html` in any browser

## Tech Stack
HTML · CSS · JavaScript · OpenWeatherMap API (Fetch)
